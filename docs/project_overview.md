# AI Healthcare Content Automation Platform

## Overview

This project is an AI-powered content automation system designed for healthcare technology companies.

The platform automatically monitors healthcare blogs, extracts newly published articles, generates LinkedIn-ready social media content using AI, creates contextual marketing images, stores generated content in Airtable, and publishes directly to LinkedIn company pages.

## Problem Statement

Healthcare organizations publish valuable content but often spend significant time manually creating social media posts and marketing assets.

The goal of this project is to automate the entire content repurposing workflow.

## Solution

The workflow continuously monitors RSS feeds and performs:

1. Blog detection
2. Content extraction
3. AI content generation
4. AI image generation
5. Airtable storage
6. LinkedIn publishing

## Tech Stack

* Make.com
* Google Gemini
* Airtable
* LinkedIn API
* RSS Feeds
* HTTP Integrations

## Business Impact

* Reduces manual content creation effort
* Improves publishing consistency
* Accelerates social media workflows
* Enables scalable content marketing
