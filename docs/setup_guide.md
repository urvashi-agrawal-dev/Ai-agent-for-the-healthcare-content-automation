# Setup Guide

## Prerequisites

* Make.com account
* Google Gemini API Key
* Airtable account
* LinkedIn Company Page
* LinkedIn Developer App

## Airtable Setup

Create a table with:

* Title
* Link
* AI Post
* Image URL
* Created Time

## Make.com Workflow

Step 1:
RSS Module

* Monitor healthcare blog RSS feed

Step 2:
Gemini Module

* Generate LinkedIn content

Step 3:
Image Generation Module

* Generate marketing image

Step 4:
Airtable Module

* Store generated content

Step 5:
LinkedIn Module

* Publish image and text post

## Testing

Publish a new blog article and verify:

* RSS detects article
* Content generated
* Image generated
* Airtable updated
* LinkedIn post published
